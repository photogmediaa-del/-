# 🏛️ نظام بلدية الكرمة المتكامل

## 📋 نظرة عامة

نظام إدارة بلدية متكامل يربط بين:
- **نظام المواطنين** (Citizen Portal)
- **نظام الإدارة** (Admin Dashboard)

جميع البيانات مرتبطة عبر `localStorage` ويتم التحديث فوراً بين النظامين.

---

## 🎯 الربط بين النظامين

### 📊 البيانات المشتركة في localStorage:

```javascript
// 1. سجل دخول المواطنين
citizenLoginLogs: [
  {
    id, name, phone, provider, 
    loginTime, logoutTime, status
  }
]

// 2. الجلسة النشطة
citizenSession: {
  name, phone, provider, 
  loginTime, status
}

// 3. الطلبات من الخدمات
citizenRequests: [
  {
    id, ref, service, name, phone,
    subject, details, address, 
    images, status, date, buildingData
  }
]

// 4. المدفوعات الإلكترونية
citizenPayments: [
  {
    id, ref, type, typeLabel,
    name, phone, amount, total,
    referenceNumber, notes,
    cardName, cardLast4,
    status, date
  }
]

// 5. الأخبار (يديرها الأدمن)
municipalityNews: [
  {
    id, title, content, date
  }
]

// 6. معلومات الاتصال (يديرها الأدمن)
municipalityContact: {
  phone, website, email, address
}
```

---

## 🔗 كيفية الربط

### 1️⃣ **المواطن يدخل للنظام:**
```
نظام المواطنين → يحفظ في citizenLoginLogs
                ↓
    نظام الإدارة → يعرض في "إدارة المواطنين"
```

### 2️⃣ **المواطن يقدم طلب:**
```
نظام المواطنين → اختيار القسم (بيئة/حدائق/بناء/تنظيم)
                → تعبئة النموذج
                → يحفظ في citizenRequests
                ↓
    نظام الإدارة → يظهر في "إدارة الطلبات"
                → الأدمن يوافق/يرفض
```

### 3️⃣ **المواطن يدفع:**
```
نظام المواطنين → اختيار نوع الدفع
                → إدخال البيانات
                → يحفظ في citizenPayments
                ↓
    نظام الإدارة → يظهر في "إدارة المدفوعات"
```

### 4️⃣ **الأدمن ينشر خبر:**
```
نظام الإدارة → إضافة خبر جديد
             → يحفظ في municipalityNews
             ↓
نظام المواطنين → يعرض الخبر فوراً في صفحة الأخبار
```

### 5️⃣ **الأدمن يعدل معلومات الاتصال:**
```
نظام الإدارة → تحديث الهاتف/الإيميل/العنوان
             → يحفظ في municipalityContact
             ↓
نظام المواطنين → يعرض المعلومات الجديدة في "اتصل بنا"
```

## 🚀 طريقة الاستخدام

### للمواطن:

1. افتح `citizen-system/index.html`
2. أدخل الاسم الثلاثي + رقم الهاتف
3. أدخل كود OTP (الرمز التجريبي: `1234`)
4. استخدم الخدمات:
   - تقديم طلبات
   - الدفع الإلكتروني
   - قراءة الأخبار
   - معلومات الاتصال

### للإدارة:

1. افتح `admin-system/index.html`
2. أدخل:
   - اسم المستخدم: `admin`
   - كلمة المرور: `admin123`
3. ستفتح لوحة التحكم `dashboard.html` تلقائياً
4. استخدم القوائم:
   - 📊 لوحة المعلومات (إحصائيات)
   - 👥 إدارة المواطنين (من دخل)
   - 📋 إدارة الطلبات (قبول/رفض)
   - 💳 إدارة المدفوعات (عرض)
   - 📰 إدارة الأخبار (نشر/حذف)
   - 📞 معلومات الاتصال (تعديل)

## 📊 لوحة التحكم - الصفحات

### 1. لوحة المعلومات (Dashboard)
```
الإحصائيات:
- إجمالي المواطنين
- المتصلين الآن
- إجمالي الطلبات
- إجمالي المدفوعات

آخر النشاطات:
- أحدث الطلبات
- أحدث المدفوعات
```

### 2. إدارة المواطنين
```
جدول يعرض:
- الاسم
- رقم الهاتف
- الشركة (زين/آسيا/كورك)
- وقت الدخول
- الحالة (متصل/غير متصل)
```

### 3. إدارة الطلبات
```
جدول يعرض:
- رقم الطلب
- القسم (بيئة/حدائق/بناء/تنظيم)
- اسم المواطن
- الموضوع
- التاريخ
- الحالة

إجراءات:
- ✅ قبول الطلب
- ❌ رفض الطلب
```

### 4. إدارة المدفوعات
```
جدول يعرض:
- رقم العملية
- نوع الدفع
- اسم المواطن
- المبلغ
- التاريخ
- الحالة (مكتمل)
```

### 5. إدارة الأخبار
```
نموذج إضافة:
- عنوان الخبر
- محتوى الخبر
- زر "نشر الخبر"

قائمة الأخبار:
- عرض كل الأخبار المنشورة
- زر "حذف" لكل خبر
```

### 6. معلومات الاتصال
```
نموذج تعديل:
- رقم الهاتف
- الموقع الإلكتروني
- البريد الإلكتروني
- العنوان
- زر "حفظ التعديلات"
```
