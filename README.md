## 🚀 طريقة الاستخدام

### للمواطنين:

1. **افتح:** `citizen-system/index.html`

2. **أدخل البيانات:**
   - الاسم: **أحمد محمد علي**
   - اختر الشركة: **زين**
   - الرقم: **07801234567**

3. **أدخل رمز التحقق:** `1234`

4. **استخدم الخدمات:**
   - اختر خدماتنا
   - اختر القسم
   - املأ النموذج
   - أرفق الصور
   - أرسل الطلب

---

### للإدارة:

#### الطريقة الأولى - لوحة الإدارة الأصلية:
1. افتح `admin-system/index.html`
2. دخول: `admin` / `admin123`
3. استخدم النظام كالعادة

#### الطريقة الثانية - مراقبة المواطنين:
1. افتح `admin-system/citizen-monitor.html`
2. شاهد:
   - من متصل الآن
   - جميع سجلات الدخول
   - جميع الطلبات المقدمة
   - الصور المرفقة
